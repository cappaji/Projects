WINDOWS EXECUTION

TO RUN IN COMMAND PROMPT

java -jar Account.jar


TO IMPORT COMPILE & RUN

1. Goto "Sourcecode" folder that is "src"
2. compile with
	javac *.java
3. run with
	java DriverClass

TO EXECUTE IN ECLIPSE 
1. Import project "CS-586_Account_Project"
2. Run the Program. (Select Driver Class as point of execution)
